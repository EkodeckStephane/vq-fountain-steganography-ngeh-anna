# Legacy ETHEGAN Status

Status: superseded by the VQ-Fountain Stega direction.

ETEHGAN remains in the workspace to preserve experiments, code, and negative
evidence. It should not be used as the active Q1 contribution without a new
technical breakthrough.

## Why It Was Deprecated

Critical limits:

1. The sender used a cover image, so strict coverless claims were not supported.
2. JPEG, blur, and resize tests failed exact recovery under the evaluated
   attack-aware probe.
3. Lightweight residual/LSB steganalysis detected the stego distribution above
   chance.
4. The evidence base was too small for a Q1 robustness or security claim.
5. The novelty was fragile if framed as GAN-based or coverless steganography.

Useful retained value:

1. exact-recovery accounting;
2. effective payload accounting;
3. attack matrices and recorded decode outcomes;
4. steganalysis sanity checks;
5. reproducibility discipline.

## Rule

Use ETHEGAN only as historical context, baseline machinery, or negative
evidence. The active manuscript direction is VQ-Fountain Stega.
